# Notice

Lizhi Illustrations is an AI-agent skill for generating hand-drawn concept-analysis illustrations, adapted from [Ian Xiaohei Illustrations](https://github.com/helloianneo) created by Ian.

The example images in `examples/images/` and `lizhi-illustrations/assets/examples/` are generated with this adapted skill and reflect the Lizhi visual language. The original visual style and workflow framework come from Ian's project.

This adapted skill uses `Lizhi` as its active recurring character. When redistributing or adapting this repository, please provide attribution to Ian for the original visual style and workflow framework.
